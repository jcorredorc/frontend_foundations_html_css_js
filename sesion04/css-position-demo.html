Bonus: position: sticky

    

      sticky es un híbrido entre relative y fixed. El elemento se comporta como relative hasta que el scroll alcanza el umbral que defines (por ejemplo top: 0), y entonces se "pega" como si fuera fixed, pero solo dentro de su contenedor padre.

Resumen rápido

    
Para tener a mano cuando estés decidiendo qué position usar.


    

      

        
static

        
Valor por defecto. Sigue el flujo normal. Ignora top/left/etc.

      

      

        
relative

        
Se desplaza desde su posición original, dejando un hueco. Ancla para hijos absolute.

      

      

        
absolute

        
Sale del flujo. Se ancla al ancestro posicionado más cercano.

      

      

        
fixed

        
Sale del flujo. Se ancla al viewport. Sobrevive al scroll.

      

      

        
sticky

        
Híbrido: relative hasta tocar el umbral, después fixed dentro de su padre.
